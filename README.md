A clone of good reads which tracks the books read by the user, record the progress and provide ratings for the same. Stores and handles large amount of data of every book ever published in the world and expose it as a catalog.
Technologies Used : Spring Boot, Apache Cassandra, Spring Data Cassandra, Spring Security, REST API, Thymeleaf.
